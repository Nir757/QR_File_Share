# Documentation

This folder contains all documentation files for the QR File Share project.

## Documentation Files

- **P2P_SETUP.md** - Setup guide for P2P outside-LAN support (for developers/users setting up the app)
- **INTEGRATION_EXAMPLE.md** - Code examples for integrating the signaling client
- **QUICK_START_P2P.md** - Quick 5-minute setup guide
- **P2P_IMPLEMENTATION_SUMMARY.md** - Technical implementation overview

## For Users

If you're setting up the app for the first time, start with:
1. **README.md** (in root) - Main project documentation
2. **QUICK_START_P2P.md** - Quick setup guide

## For Developers

If you're integrating or modifying the code:
1. **INTEGRATION_EXAMPLE.md** - Code integration examples
2. **P2P_SETUP.md** - Detailed setup instructions
3. **signaling-server/README.md** - Signaling server API documentation
